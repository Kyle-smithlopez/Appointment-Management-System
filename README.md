# AppointmentManagementSystem
Title: Appointment Management System

This is an appointment scheduler application. It is a simple application that allows the user to schedule appointments. The user can add, edit, and delete appointments. The user can also view the appointments by month or week. The user can add, edit, and view customers.

Author: Kyle Smith Application Version: 1.0 Date: 1/7/2023

IDE version:IntelliJ IDEA 2022.3.1 (Community Edition) Build #IC-223.8214.52, built on December 20, 2022 Runtime version: 17.0.5+1-b653.23 amd64 JDK version: 17.0.5 JavaFX version: 17.0.2-ea

How to run the application: Launch Application brings you to the login screen. Login using valid credentials will bring you to the main screen. You can navigate to Appointments, Customers or Reports. In Appointments, You can view appointments by month or week. You can also add, edit, and delete appointments. From the Customers screen, you can view a table of customers within the database. You can also add, edit, and view customers. In reports, You will be presented with three reports. You can navigate back to the main screen by clicking the back button.

Additional Report: The third report is located on the reports screen. It provides a count of customers by each division. MYSQL CONNECTION DRIVER VERSION: 8.0.31
